ARQUITECTURA CLIENTE/SERVIDOR







ejemplos de cliente-servidor

Un ejemplo de una red cliente-servidor es el uso del servicio de correo electrónico de un ISP para enviar, recibir y almacenar correo electrónico.
El cliente de correo electrónico en una PC doméstica emite una solicitud al servidor
de correo electrónico del ISP para que se le envíe todo correo no leído.


algunos ejemplos mas serian: navegador webs, Protocolo FTP, SSH, juegos online, sistema DNS, correo movil
